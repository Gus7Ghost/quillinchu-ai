# Misión Científica del LabIAR - Quillinchu AI

Establecer la base científica y tecnológica para el seguimiento autónomo, robusto y en tiempo real de personas utilizando drones. El software desarrollado debe garantizar:
- Alta precisión en el seguimiento.
- Respuesta en tiempo real minimizando la latencia.
- Seguridad de vuelo ante cualquier contingencia.
